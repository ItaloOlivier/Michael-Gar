How to Add News

  1. Fill out the form above with your article details
  2. Click "Generate JSON" to create the article data
  3. Copy the generated JSON
  4. Open news.json in your repository
  5. Add a comma after the last article's closing }
  6. Paste your new article before the final ]
  7. Commit and push the changes to GitHub

Adding Images

  1. Upload your image to /images/news/ folder in the repository
  2. Use the path images/news/your-image.jpg in the Image URL field
  3. Or use an external image URL from another website

Adding YouTube Videos

  1. Go to the YouTube video you want to embed
  2. Copy the full URL from your browser (e.g., https://www.youtube.com/watch?v=ABC123)
  3. Paste it in the YouTube URL field
